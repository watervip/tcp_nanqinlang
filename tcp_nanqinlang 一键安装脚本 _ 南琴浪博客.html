tcp_nanqinlang 一键安装脚本

基于拥塞控制技术 tcp-bbr (在此由衷感谢各位开发者大佬们)
清真 膜改 BBR by 南琴浪
一键脚本 tcp_nanqinlang.sh
适用于 Debian / Ubuntu
releases: https://github.com/nanqinlang/tcp_nanqinlang/releases

脚本用法说明

command : { install | start | status | upgrade | uninstall }
根据运行的 command ，会输出 logfile: /home/tcp_nanqinlang_{commmand}.log

  • install 安装内核
    此命令用于安装指定内核版本，若 您的当前内核 已是 v4.9.3 及以上（例如 debian 9），可略过此步 直接 start
    • 运行 install 过程中 会在当前目录下载三个deb: image / headers_all / headers_bit
      脚本运行过程中 生成的文件 除了日志输出位于 /home 外，其他全部均位于 /home/tcp_nanqinlang 目录
    • 运行 install 过程中 会在 /etc/sysctl.conf 中添加三项设置: fastopen/fq_codel/tcp_nanqinlang
    • 若选择安装内核版本4.10.2 则不需再运行 start command，install 完成后直接reboot就行，可用 status command进行验证
      确认后 reboot
      重启开机后 bash start
  • start 启用算法
    此命令用于编译并启用算法
    • 也可使用此命令 在自己使用的内核版本上 编译生成tcp_nanqinlang.ko文件并加载
    • 若出现报错 {error} making mod failed, please check! ，请检查 gcc-4.9 是否已安装并被优先引用
  • status 运行状态
    此命令用于检查 tcp_nanqinlang 是否已被 加载(install) 和 启用(running)
  • upgrade 检测更新
    此命令用于将 系统内核版本 升级至最新
  • uninstall 停止算法
    此命令用于 卸载 算法
    运行 uninstall 过程后 并不会删除已安装的内核，仅 移除 sysctl.conf的tcp_nanqinlang，then
    reboot 后模块才会停止运行

本浪的膜改参数

请注意

  1. 运行脚本需使用 bash 命令，不可直接 ./xxx.sh 运行
  2. 若选择内核版本为 v4.10.2，install 完成后不需要再 start
  3. 卸载命令不会更改您的内核版本
